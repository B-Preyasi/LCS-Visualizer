# LCS Visualizer (Java Swing)

This project visualizes the **Longest Common Subsequence (LCS)** algorithm using **Java Swing GUI**.

---

## 📌 Features

* Dynamic Programming Table visualization
* Direction table with arrows (↖ ↑ ←)
* Step-by-step animation
* Backtracking to display LCS

---

## 🧠 Algorithm Used

* Dynamic Programming
* Time Complexity: O(n × m)
* Space Complexity: O(n × m)

---


## ▶️ How to Run

1. Compile the code:

   ```
   javac LCS.java
   ```
2. Run:

   ```
   java LCS
   ```

---

## 💡 Example

Input:

```
String 1: ABCB
String 2: BDCAB
```

Output:

```
LCS: BCB
Length: 3
```
